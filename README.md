# 🔐 BreachGuard

BreachGuard is a simple web application that checks whether an email address has been involved in a data breach.

## 🚀 Features
- 🔍 Check if an email is breached
- 🎨 Clean and modern UI
- ⏳ Loading animation while checking
- 🟢 Safe / 🔴 Breach result with color indicators
- 🔒 Security tips shown if breach is detected

## 🛠️ Tech Stack
- Python (Flask)
- HTML, CSS, JavaScript
- Render (Deployment)

## 📸 Demo
Live Project: https://breachguard-onm2.onrender.com

## 💡 How it Works
- User enters email
- Backend processes request
- Result displayed with UI feedback
- Security tips shown if needed

This project is for educational purposes and demonstrates cybersecurity awareness.
